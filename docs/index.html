The problem

  • AI tools work best with real context — but raw copy-paste leaks sensitive data.
  • Manual redaction is slow and error-prone.
  • Privacy/compliance teams need on-device processing.

The DataMask approach

  • Local-only: processing happens in your browser; no network calls.
  • Your dictionary: define rules like “ACME” → “customer name”.
  • Two actions: Anonymize & Copy Selection or entire Page.
  • Private by design: no analytics, no remote code, no telemetry.
